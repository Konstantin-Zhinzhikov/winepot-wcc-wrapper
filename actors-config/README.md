Для запуска необходима переменная окружения "APIFY_TOKEN"
При тестовом запуске её можно положить в файл `<repository_root>/apify-config/.env`

Использование: 
1. `npm install`
2. `npm run upload`

В файле `<repository_root>/apify-config/config.json` прописаны названия KVstores в Apify, куда нужно отправить и сохранить настройки.
